how to edit the KGI homepage

if you must

  1. Open your file explorer and navigate to S:\ContentDM\ResponsiveSiteFiles\KGIresponsive
    1. right-click on home.html and choose Edit with Notepad++
    2. you should see something like this (fonts and colors will vary):
    3. make the necessary changes
    4. add one(1) to the version number at the very top of the file:
    5. save!
    6. leave the file explorer open
    next step
  2. Open your web browser and navigate to the ContentDM website configuration tool
    1. log in
    2. change the tealish-colored box at the top of the page to say Responsive instead of 6.x Website
    3. on the left-hand side of the page, click Page Types and then Home Page
    4. scroll to the bottom of the page
    5. next to the field for Custom html, click the browse button
    6. the file upload window should come up:
    7. drag home.html over from the file explorer
    8. it will automatically replace the previous version
    9. all done!
